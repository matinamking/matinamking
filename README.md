## 🔧 نکات فنی — نحوهٔ فعال‌سازی آمار خصوصی (اختیاری)

اگر خواستی آمار private هم نشان داده شود، باید نسخهٔ خودِ سرویس `github-readme-stats` را دیپلوی کرده و توکن گیت‌هاب را در محیط Vercel/Netlify قرار بدهی. خلاصهٔ مراحل:

1. فول‌فورک کن یا clone کن `github-readme-stats` repo به حساب گیت‌هابت.
2. روی Vercel یک پروژه جدید بساز و ریپو را متصل کن.
3. در تنظیمات Vercel، متغیر محیطی `GITHUB_TOKEN` را با یک Personal Access Token که `repo` scope دارد تنظیم کن.
4. آدرس API خودت را در README با `https://<your-vercel>.vercel.app/api?username=matinamking&count_private=true&include_all_commits=true` استفاده کن.

> هشدار: به دلایل امنیتی، اگر توکن را عمومی در README قرار دهی، خطرناک است — حتماً از secrets/Environment variables در Vercel استفاده کن.
